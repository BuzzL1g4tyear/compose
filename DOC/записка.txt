создать одного главного админа (ГА), который может добавлять других участников(админы, работники)

изначально создан один ГА, который впрваве добавлять новых юзеров

Процесс добавления

ГА создаёт одного админа магазина(АМ) -> АМ добавляет, по номеру телефона, участников 
	-> Добавленный участник при регистрации указывает собственный номер -> Пользователь добавлен.
